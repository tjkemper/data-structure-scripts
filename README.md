# Data Structure Scripts

Python scripts leveraging data structures

### Data Structures
* Max Heap

### Setup

```
virtualenv venv
. venv/bin/activate
pip install --editable .
```

### Teardown

```
deactivate
```

### `maxheap` usage

**Return max heap.**

```
maxheap input1.txt
```

**Pretty print max heap.**

```
maxheap input1.txt --pretty-print
```

**Return sorted list (descending).**

```
maxheap input1.txt --sort
```

**More information.**

```
maxheap --help
```
